Showing HTML on the desktop as wallpaper.

Features:
*Listening mouse middle key. Click to hide/show the desktop icons.
*Passing mouse left key move/click signals to the website.
*Adding some crash handlers
*Multi-screen support

Based on:https://github.com/ThomasHuai/Wallpaper

For demo only.To put into usage, please add necessary restrictions.

Best wishes.


在桌面显示HTML页面。

特性:
*监听鼠标中键。点击以隐藏/显示桌面图标。
*将鼠标左键移动/点击消息传递给HTML页面
*增加部分错误处理
*支持多屏幕

基于https://github.com/ThomasHuai/Wallpaper

仅仅用于演示，实际使用请合理配置权限.
